# Bookmark App with Remix + Cloudflare Stacks

- Remix
  - Drizzle ORM, Drizzle Kit
- Cloudflare Pages
- KV
- D1
- Queue
- R2
- Cron Triggers

## Development

### Migration

```sh
npm run migrations:gen
```

### Start server

```sh
npm run dev
```

Open up [http://127.0.0.1:8788](http://127.0.0.1:8788) and you should be ready to go!

## Deployment

### Deploy

```sh
pages:deploy
```
